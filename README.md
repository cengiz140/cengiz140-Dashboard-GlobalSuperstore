# Global Superstore Sales Dashboard

## Overview
This project is a Power BI dashboard built using the Global Superstore dataset. I created it to explore sales performance, profit, and customer behaviour, and to practice turning data into actionable insights.



## What this project looks at
- Sales and profit across different regions  
- Trends over time  
- Product and category performance  
- Customer segments and buying patterns
- Operations analysis



## Tools used
- Power BI  
- Excel  
- SQL (where needed)



## Key takeaways
- Sales tend to increase towards the end of the year  
- Some regions bring in high sales but not much profit 
- A small number of products account for a large share of revenue  
- Customer segments behave quite differently in terms of spending
